# goodix-factory-tools

这是一个基于 Mongoose + 纯 C 实现的通信后端，与 HTML/JS 前端分离的架构方案。主要用于 GH3x2x 生物传感器的自动化检测、交互和数据采样。

### 近期更新：高级控制功能 (Advanced Control)
相较于早期的基础读写协议层，当前版本我们在不需要原 C++ `cardiff_rpc` 库的前提下，新增补齐并打通了以下链路：
1. **模式选择**：支持 `Mcu Online Work Mode` 与 `Auto Pass Through Mode` (发送 `0x10` 指令)。
2. **硬件版本读取**：一键触发查询板卡版本号 (`0x19` 指令)。
3. **芯片控制**：WAKE_UP / SLEEP / RESET / RESUME 状态机控制 (`0x17` 指令)。
4. **灵活寄存器组**：新增独立的 “读寄存器” 和 “写寄存器” 单卡槽，对 `0x03` 进行点对点测试读取。
5. **智能 `.config` 配置导入**：浏览器 Web 端原生支持读写 `.config` / `.txt` 等配置文件，前端将直接正则匹配提取有效的类似于 `{0x0016, 0x001F}` 的地址和值组合，以 UI 列表展示并允许“一键批量发送”。

详细说明请参考：[开发与测试手册](docs/README.md)
